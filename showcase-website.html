Website Development Techs

The main technology I used for developing website application (other than native html for small projects) are React JS and Next JS. React provides a lot of conveniences in developing a website application and is proving to be the most popular website framework. Using React, I have developed 3 websites for work and 2 other websites for personal project.

Styling using Bootstrap and Tailwind CSS.

Styling has to be one of the most important aspects required in the process of developing a good website, styling includes many things such as responsiveness, color palette, typography, element consistency, etc. When it comes to styling I am quite experienced in using plain CSS bootstrap , and tailwind CSS.

Third Party Integration

In my experience of developing websites, there are a few unique client or stakeholder request that requires the integration of third party service / plugins like:

  • - Twilio Web Real Time Connection (RTC)
  • - Socket IO (Web Socket for real time graph / data stream)
  • - React Big Calendar (calendar scheduling feature)
  • - React QR Scanner (Webcam QR Scanner)
  • - Gqlgen (GraphQL typeSafe API)
